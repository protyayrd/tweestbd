Admin Navigation Check

This page provides instructions to check if the "Reviews & Ratings" navigation item is properly displayed in the admin sidebar.

Step 1: Open Browser Console

First, open your browser's developer console:

  • Chrome: Press F12 or Ctrl+Shift+J (Windows/Linux) or Cmd+Option+J (Mac)
  • Firefox: Press F12 or Ctrl+Shift+K (Windows/Linux) or Cmd+Option+K (Mac)
  • Safari: Enable Developer Tools in Preferences â†’ Advanced, then press Cmd+Option+C

Step 2: Navigate to Admin Panel

Go to the admin panel at: /admin

Step 3: Check Console Logs

In the browser console, look for these log entries:

  • AdminSidebar rendering, navigation items: [...]
  • Rendering nav item: Dashboard
  • Rendering nav item: Products
  • Rendering nav item: Orders
  • Rendering nav item: Payments
  • Rendering nav item: Customers
  • Rendering nav item: Reviews & Ratings
  • Rendering nav item: Bulk Orders
  • etc.

Verify that "Reviews & Ratings" is in this list.

Step 4: Inspect the DOM

In the browser's developer tools, select the Elements/Inspector tab and look for the sidebar navigation list.

Find the element with text "Reviews & Ratings" to confirm it exists in the DOM.

Step 5: Check for CSS Issues

If the item exists in the DOM but isn't visible, there might be CSS issues hiding it. Check for:

  • Elements with display: none
  • Elements positioned off-screen
  • Invisible text (color matching background)

Step 6: Try Direct Navigation

Try navigating directly to the Reviews page: /admin/reviews

If this works, the issue is just with the sidebar display.
